**Summary**
- Reports whether message-skip mode is currently active in the UI.

**Tags**
- ui

**Syntax**
- `MESSKIP()`

**Signatures / argument rules**
- `MESSKIP()` → `long`

**Arguments**
- (none)

**Semantics**
- Returns `1` if message-skip mode is active, otherwise returns `0`.

**Errors & validation**
- (none)

**Examples**
