**Summary**
- Returns the minimum of one or more integers.

**Tags**
- math

**Syntax**
- `MIN(x [, y ...])`

**Signatures / argument rules**
- `MIN(x [, y ...])` → `long`
  - Requires at least 1 argument.

**Arguments**
- `x` (int)
- `y...` (optional, int)

**Semantics**
- Returns the minimum value among all provided arguments.

**Errors & validation**
- (none)

**Examples**
- `MIN(1)` returns `1`.
- `MIN(1, 5, 2)` returns `1`.
