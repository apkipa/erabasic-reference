**Summary**
- Reports whether the console is currently in an auto-redraw mode.

**Tags**
- ui

**Syntax**
- `CURRENTREDRAW()`

**Signatures / argument rules**
- `CURRENTREDRAW()` → `long`

**Arguments**
- (none)

**Semantics**
- Returns `0` if redraw mode is off, otherwise returns `1`.

**Errors & validation**
- (none)

**Examples**
- `isRedrawing = CURRENTREDRAW()`

**Progress state**
- complete
