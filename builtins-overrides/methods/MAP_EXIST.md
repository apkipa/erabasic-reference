**Summary**
- Checks whether a named map exists.

**Tags**
- map
- data-structures

**Syntax**
- `MAP_EXIST(mapName)`

**Signatures / argument rules**
- `MAP_EXIST(mapName)` → `long`

**Arguments**
- `mapName` (string): map identifier.

**Semantics**
- Returns `1` if the map exists.
- Returns `0` otherwise.

**Errors & validation**
- None.

**Examples**
- `IF MAP_EXIST("session")`

**Progress state**
- complete
