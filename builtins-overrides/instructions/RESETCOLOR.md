**Summary**
- Resets the current text color to the configured default foreground color.

**Tags**
- ui

**Syntax**
- `RESETCOLOR`

**Arguments**
- None.

**Semantics**
- Sets the current text color to the configured default (config item `ForeColor`).
- Does not print output.

**Errors & validation**
- (none)

**Examples**
- `RESETCOLOR`

**Progress state**
- complete
