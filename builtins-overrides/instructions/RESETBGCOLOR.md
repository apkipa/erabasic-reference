**Summary**
- Resets the current background color to the configured default background color.

**Tags**
- ui

**Syntax**
- `RESETBGCOLOR`

**Arguments**
- None.

**Semantics**
- Sets the current background color to the configured default (config item `BackColor`).
- Does not print output.

**Errors & validation**
- (none)

**Examples**
- `RESETBGCOLOR`

**Progress state**
- complete
