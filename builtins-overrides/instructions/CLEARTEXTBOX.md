**Summary**
- Clears the main output text box (removes all currently displayed lines).

**Tags**
- ui

**Syntax**
- `CLEARTEXTBOX`

**Arguments**
- None.

**Semantics**
- Clears the UI’s main output area.
- This instruction still executes even when output skipping is active.

**Errors & validation**
- (none)

**Examples**
- `CLEARTEXTBOX`

**Progress state**
- complete
