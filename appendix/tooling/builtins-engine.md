# Built-ins Extracted from Engine Source (`emuera.em`)

This appendix is **engine-source-of-truth**: it lists built-in *instruction keywords* and built-in *expression-function (method) names* extracted from the `emuera.em` codebase in this workspace.

Notes:
- Instruction keywords are the keys in `FunctionIdentifier.GetInstructionNameDic()` (registered via `addFunction(FunctionCode.X, ...)` etc.).
- Expression-function names are the keys in `FunctionMethodCreator.GetMethodList()` (registered in `Creator*.cs`).
- `IgnoreCase` affects how keywords are matched at runtime, but this file lists the canonical spellings as written in code.
- Some parser behaviors use internal helpers that are **not** resolvable via the normal instruction dictionary (notably the internal `SET` used for assignment statements).

Counts (as extracted):
- Instructions: 302
- Expression functions: 266
